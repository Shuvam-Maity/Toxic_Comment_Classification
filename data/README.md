# Data

This project uses the [Jigsaw Toxic Comment Classification Challenge](https://www.kaggle.com/competitions/jigsaw-toxic-comment-classification-challenge) dataset from Kaggle.

Kaggle competition data can't be redistributed, so it isn't included in this repo. To run the notebooks:

1. Join the competition on Kaggle (free) and download `train.csv`.
2. Place it here as `data/train.csv`.

Or, with the Kaggle CLI:

\`\`\`bash
kaggle competitions download -c jigsaw-toxic-comment-classification-challenge -p data/
unzip data/jigsaw-toxic-comment-classification-challenge.zip -d data/
\`\`\`
